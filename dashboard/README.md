# Estonia driving exams dashboard

Next.js dashboard for exploring Estonian driving exam statistics.

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start development server |
| `npm run import-data` | Import CSV files into SQLite |
| `npm run build` | Import data and build for production |
| `npm run start` | Run production server |

## Stack

- Next.js 16, React 19, Tailwind CSS 4
- Kysely + better-sqlite3
- Recharts
